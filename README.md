COVID-19 India Trend Analysis and Forecasting

Project Overview

This project analyzes COVID-19 cases in India using data analysis and time-series forecasting. It provides insights into confirmed, recovered, and death cases through visualizations and predicts future trends using the Prophet forecasting model.

Technologies Used

- Python
- Pandas
- Plotly
- Prophet
- NumPy

Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Daily and cumulative case analysis
- Interactive visualizations
- 7-day COVID-19 case forecasting
- Trend analysis for confirmed, recovered, and death cases

Dataset

- COVID-19 India Dataset (Kaggle)

Project Workflow

1. Load and clean the dataset.
2. Analyze daily COVID-19 trends.
3. Calculate growth rates and statistics.
4. Create interactive charts using Plotly.
5. Forecast future cases using Prophet.
6. Interpret and summarize the results.

Results

- Identified trends in confirmed, recovered, and death cases.
- Visualized daily and cumulative case patterns.
- Generated a 7-day forecast using the Prophet model.
- Produced interactive dashboards for better data interpretation.

Future Improvements

- Add state-wise analysis.
- Build a Streamlit web application.
- Include vaccination and testing data.
- Extend forecasting to 30-day and 90-day predictions.
